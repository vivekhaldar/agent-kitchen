Not another orchestrator. A dashboard that makes sense of every session your AI agents have run — across repos, your home directory, notes, configs, everywhere.

The gap

Orchestrators assume your agent lives in a repo. Yours doesn't.

Conductor & Sculptor

C Every session scoped to a single git repository
C Agents run in containers or worktrees — isolated from your machine context
C Must launch agents through their UI to get any visibility
C No view of non-repo work: dotfiles, notes, home directory tasks

Agent Kitchen

AK Sessions grouped by repo and non-repo locations like ~/dotfiles, ~/notes
AK Agents run natively with full access to your memories, skills, configs
AK Works retroactively — scans sessions you already ran, however you ran them
AK AI-generated summaries tell you what happened, not just which file changed
